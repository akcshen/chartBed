# chartBed

chartBed 是一个轻量级的图床项目，旨在提供简单、高效的图片存储与管理服务。用户可以上传图片并获取可直接使用的图片链接，支持便捷的图片管理与分享。

## 功能特性

- **简单易用**：上传图片后立即生成分享链接。
- **高效存储**：使用高效的图像存储解决方案，保证图片上传与加载速度。
- **便捷管理**：提供简单的图片管理界面，方便用户查看、删除上传的图片。
- **支持多种格式**：支持多种常见图片格式（JPEG, PNG, GIF 等）。
